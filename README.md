# Book store

Разработка книжного магазина

## Установка

1. Клонирование репозитория

```git clone https://github.com/Lirgat/book-store```

2. Переход в директорию проекта

```cd book-store```

3. Установка npm 

```npm install```

4. Сборка проекта

```npm run build ```

## Стек проекта:

### Основной стек:

HTML5, CSS3, JS(ES6), TypeScript, SCSS

### Сборщик: 

Webpack 5

#### Плагины Webpack:

1. TerserPlugin
2. HtmlWebpackPlugin

#### Лоадеры:

1. css-loader
2. style-loader
